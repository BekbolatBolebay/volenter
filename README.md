# Alem AI еріктілер платформасы

Еріктілер қызметін жұмыс ретінде емес, қызықты ойын және болашақ мансабыңызға қуатты инвестиция ретінде сезініңіз.

## 🚀 Шолу

**Alem AI** — еріктілер тәжірибесін түбегейлі өзгертуге арналған заманауи платформа. Геймификация мен жасанды интеллектті біріктіре отырып, біз әлеуметтік әсерді еріктілер үшін де, әкімшілер үшін де пайдалы саяхатқа айналдырамыз.

## ✨ Негізгі мүмкіндіктер

### Еріктілер үшін
- **Геймификацияланған тәжірибе:** Дағдыларыңызды шыңдаңыз, белгілер (badges) алыңыз және XP (тәжірибе ұпайларын) жинаңыз.
- **Сыйақылар:** Жинаған XP ұпайларыңызды шынайы өмірдегі сыйлықтар мен жеңілдіктерге айырбастаңыз.
- **Дағдылар ағашы:** Өз дамуыңызды қадағалап, кәсіби құзыреттіліктеріңізді арттырыңыз.

### Әкімшілер үшін
- **ЖИ арқылы тапсырмалар жасау:** Жасанды интеллект көмегімен еріктілер тапсырмаларын тиімді жасаңыз және басқарыңыз.
- **Автоматты сәйкестендіру:** ЖИ-ге жобаларыңыз үшін мінсіз еріктілерді табуға мүмкіндік беріңіз.
- **Алаяқтықты анықтау:** Gemini Vision көмегімен фотоесептерді автоматты түрде тексеру және алаяқтықты анықтау.

## 🛠️ Технологиялық стек

- **Фреймворк:** [Next.js](https://nextjs.org/)
- **Тіл:** TypeScript
- **Стиль:** CSS / Tailwind CSS
- **ЖИ интеграциясы:** Google Gemini
- **Анимациялар:** Framer Motion
- **Иконкалар:** Lucide React

## 🚦 Жұмысты бастау

Алдымен әзірлеу серверін іске қосыңыз:

```bash
npm run dev
# немесе
yarn dev
# немесе
pnpm dev
# немесе
bun dev
```

Нәтижені көру үшін браузерде [http://localhost:3000](http://localhost:3000) сілтемесін ашыңыз.
